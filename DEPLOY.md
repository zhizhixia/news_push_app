# 微信云托管部署说明

本文档说明了如何将新闻推送应用部署到微信云托管平台。

## 部署前准备

### 1. 环境变量配置
在微信云托管控制台中，需要配置以下环境变量：

```
DASHSCOPE_API_KEY=你的阿里云百炼API密钥
WECHAT_TOKEN=你的微信公众号Token
WECHAT_APPID=你的微信公众号AppID  
WECHAT_APPSECRET=你的微信公众号AppSecret
DEFAULT_CITY=北京
PUSH_TIME_HOUR=8
PUSH_TIME_MINUTE=0
LOG_LEVEL=INFO
HOST=0.0.0.0
PORT=8000
```

### 2. 服务配置
- **CPU**: 0.25核
- **内存**: 0.5GB  
- **端口**: 8000
- **协议**: HTTP

### 3. 健康检查配置
- **检查路径**: /health
- **检查端口**: 8000
- **初始延迟**: 30秒
- **检查间隔**: 30秒

## 部署步骤

1. 确保代码仓库中包含 `Dockerfile`
2. 在微信云托管控制台创建服务
3. 配置环境变量
4. 触发部署

## 注意事项

1. **端口配置**: 应用监听8000端口，确保与云托管配置一致
2. **环境变量**: 所有敏感信息通过环境变量配置，不要硬编码在代码中
3. **健康检查**: 应用提供 `/health` 端点用于健康检查
4. **日志**: 所有日志会输出到标准输出，可在云托管控制台查看
5. **MCP服务**: 已开通墨迹天气、城市新闻、高德地图MCP服务，无需额外配置

## 微信公众号配置

部署成功后，需要在微信公众号后台配置服务器URL：
```
https://你的云托管域名/api/wechat
```

Token验证通过后即可正常使用服务。
